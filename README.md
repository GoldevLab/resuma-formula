# Resuma Formula

WebGPU desk-toy racer on **Resuma 1.2.15**: modes **Kit / Drive / Studio**.

Inspired by [Formula on webgpu.com](https://www.webgpu.com/showcase/formula-f1-model-kit-webgl/) ([live demo](https://lab.patrickheintzmann.com/demo/demoFormula)) — **no public source** from Patrick Heintzmann; this is an original Resuma + WebGPU reimplementation.

## Run

```bash
cd resuma-formula
cargo run
# open http://127.0.0.1:3000
```

Needs a browser with WebGPU. Drive with WASD / arrows.
